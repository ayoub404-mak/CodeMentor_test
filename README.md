# 🧠 CodeMentor AI

AI-powered Python coding tutor — learn syntax, data structures & algorithms 
with personalized AI mentoring.

## Quick Start (2 minutes)

### 1. Clone & Install
```bash
git clone https://github.com/you/codementor-ai.git
cd codementor-ai
pip install -r requirements.txt
```

### 2. Run
```bash
python app.py
```

### 3. Open Browser
Go to http://localhost:8000

The app will guide you through:
→ Choosing your AI provider (Mistral or Gemini) and entering your API key
→ Taking a quick AI placement exam
→ Starting your personalized learning path

Your progress auto-saves locally. Close and reopen anytime to continue.

## Requirements
- Python 3.10+
- A Mistral AI API key (console.mistral.ai) **or** Google Gemini API key (aistudio.google.com)
- A modern web browser
